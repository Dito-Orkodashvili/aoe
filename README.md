# AOE II Georgian Community Website

A community platform for **Age of Empires II** focused on the Georgian scene — player profiles, tournaments, match history, stats, and more.

## Tech Stack

- **Next.js** (App Router / Server Actions depending on your current setup)
- **React**
- **Tailwind CSS**
- **Supabase**
    - Postgres + RLS
    - Auth
    - Storage (optional)
- AoE2 data via community APIs (World’s Edge / other sources depending on endpoints used)
